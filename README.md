# Setup

## Requirements

* Python 3.6+

## Installation

Install IJCAI18 with pip:

```sh
$ pip install IJCAI18
```

or directly from the source code:

```sh
$ git clone https://github.com/KenMercusLai/testrepo.git
$ cd testrepo
$ python setup.py install
```

# Usage

After installation, the package can imported:

```sh
$ python
>>> import IJCAI18
>>> IJCAI18.__version__
```
